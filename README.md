# OpenClaw API Manager (Standalone)

Extracted standalone script from `kejilion.sh` menu item **6. API管理**.

## Usage

```bash
bash openclaw-api-manager.sh
```

## One-line install to /root

```bash
curl -fsSL https://raw.githubusercontent.com/jacurtwong/openclaw-api-manager/main/openclaw-api-manager.sh -o /root/openclaw-api-manager.sh && chmod +x /root/openclaw-api-manager.sh
```
